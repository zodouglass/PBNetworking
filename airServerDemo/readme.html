Push Button Engine AIR Socket Server

Step 1: Download Push Button Engine (using 1.0 for this demo)

GitHub: https://github.com/PushButtonLabs/PushButtonEngine
Googlecode: http://pushbuttonengine.googlecode.com/files/PushButtonEngine-r958-1.0.zip

Step 2: Download Push Button Networking

Master: https://github.com/PushButtonLabs/PBNetworking
AIR Fork: https://github.com/zodouglass/PBNetworking

Step 3: Create a AIR project (using Flex 4.1 / AIR 2.6 SDK for this demo).

If you're using FlashBuilder or FlashDevelop as your IDE:
Setup a new AS3 AIR 2.0 project, and set the project to compile airServerDemo/src/ServerMain.as

If you're using Flash Professional, create a new AIR 2.0 .fla file, and set ServerMain.as as the Document Class
Add class paths paths for PBE and PB Networking src folders

Under Additional Compiler Options, make sure you add --keep-as3-metadata+=TypeHint,EditorData,Embed (Note: Flex 4.5 introduced an error where it doesnt keep the Embed metadata. More info here http://blog.natebeck.net/2011/05/flex-sdk-4-5-doesnt-keep-embed-metadata/)

Make sure your application is using AIR 2.0:
<application xmlns="http://ns.adobe.com/air/application/2.0">

Compile and run, or package and install, the AIR application.

Step 4: Extreme Multiplayer Click Demo

Setup a new Flex 4 project. The main client Application file is located under /airServerDemo/src/ClientMain.mxml
Add library paths for PBE and PBNetworking.
Compile and connect to the AIR server. By default, it is running on port 1337.
